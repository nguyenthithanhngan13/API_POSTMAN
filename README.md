# API Testing with Postman

## Giới thiệu
Dự án kiểm thử API cho các bảng:
- Categories
- Products
- Employees
- Customers
- Suppliers

## Nội dung thực hiện
- Viết test case cho các API  
- Viết script kiểm tra phản hồi của API  
- Chạy test tự động hàng loạt  
- Tạo test report  
- Thực hiện bug report  

## Hướng dẫn chạy file export
- Import file collection `.json` vào Postman
